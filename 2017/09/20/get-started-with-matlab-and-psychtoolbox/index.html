初识 MATLAB 与 Psychtoolbox

本文针对心理学的同学，初步介绍 MATLAB 与 Psychtoolbox 这两项心理学编程经常使用的工具。

MATLAB 是什么？

MATLAB 是 MathWorks 开发的一款技术计算语言。

数学·图形·编程。无论是分析数据、开发算法还是创建模型， MATLAB 都是针对您的思维方式和工作内容而设计的。

参见：MathWorks 的中文官网

如何安装 MATLAB？

获取正版软件

首先可以查看你的学校是否购买了正版全校授权资格，若有，就可以根据邮件提示，使用正版软件。

若没有，还可以购买学生版软件或下载30天试用版，然后根据提示进行安装。

此外，还可以在网络上搜索试用版或破解版进行学习。

注意版本

如果你在自己的电脑上安装 MATLAB，建议尽可能安装新的版本，因为越新的版本具有更多新的功能，且使用起来的区别并不多，如果有区别，往往是改良，而不是变糟糕。

在一些学校的机房，可能安装了较老版本的 MATLAB，不用因此过于担心程序的兼容性问题，因为兼容性的问题更多出现在 Psychtoolbox 的不同版本之间，常用的 MATLAB 自带指令并没有重大变化，如果有（如table这种数据结构在R2013b以后的版本中才出现），则应该优先考虑升级机房的老版本 MATLAB，或避免使用新功能，而不是刻意在自己的电脑上也安装老版本 MATLAB，这将阻碍你学习最新的功能。

另外请注意，Windows 系统分为32位（x86）和64位（x64）两种情况，安装 MATLAB 时要注意选择相应版本。较新的 macOS 系统皆为64位，一般不用考虑 MATLAB 版本的问题。

Psychtoolbox 是什么？

Psychtoolbox 是一套免费的 MATLAB 和 GNU Octave（用不到） 的函数集，专用于视觉和神经科学研究。它使研究者更易于合成和显示精确控制的视听刺激，并与观察者交互。

参见：Psychtoolbox 的官方网站

如何安装 Psychtoolbox？

获取

在官网下载。

注意版本

3.0.11 版是最后一个支持 32 位 MATLAB的 版本。如果电脑或 MATLAB 是 32 位的，请安装 3.0.11 版的 Psychtoolbox 。

This was the final 3.0.11 release before start of the 3.0.12 series. It was the last version that supported 32-Bit Matlab on any operating system (specifically on Linux and Windows, as support for 32-Bit Matlab for OSX was cancelled already in 3.0.10)

参见：psychtoolbox.org/versions/
